## 📱 О проекте

**IntegrationHub** — это интерактивный прототип мобильного интерфейса для управления интеграциями между корпоративными системами (Salesforce, HubSpot, Pipedrive, Shopify и др.).

Проект разработан в рамках производственной практики по профессиональному модулю **ПМ.02 «Подготовка интерфейсной графики»**.

### ✨ Особенности

- 🎨 **Дизайн-система**: единая цветовая палитра, типографика, компоненты
- 📱 **Адаптивность**: интерфейс оптимизирован под мобильные устройства
- 🌓 **Тёмная/светлая тема**: переключение с сохранением в localStorage
- 🔌 **Интерактивные элементы**:
  - Сегментированный контроль (Интеграция / Ручная)
  - Toggle-переключатели
  - Аккордеон для вложенных объектов
  - Модальные окна и уведомления
- 📦 **PWA-поддержка**: установка на главный экран, работа офлайн

### 🛠 Технологии

- **HTML5** — семантическая разметка
- **CSS3** — переменные, Flexbox, Grid, анимации
- **JavaScript (Vanilla)** — интерактивность без фреймворков
- **PWA** — manifest.json + Service Worker

## 🚀 Быстрый старт

1. Откройте [Demo](https://ВАШ_НИК.github.io/integration-hub-demo/)
2. Или запустите локально:
   - Откройте файл `index.html` в браузере.

### Установка как приложения (PWA)

**Android (Chrome):**
1. Откройте сайт в браузере
2. Нажмите меню (⋮) → **«Установить приложение»**

**iOS (Safari):**
1. Откройте сайт в Safari
2. Нажмите «Поделиться» (📤) → **«На экран "Домой"»**
